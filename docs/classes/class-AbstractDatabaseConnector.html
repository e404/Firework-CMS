Prepares an SQL query.

Prepares an SQL query.

This method makes queries safe against SQL injection attempts.

Parameters

$query

The SQL query string.
@VAL prepares a value string,
@VAR prepares a column name, or some other sort of identifier.

$field,…
The fields that should be replaced in the prepared query.

Returns

string

        SELECT *
            FROM table

        $search_value
    )
